E-Commerce Data Analysis Project

 Project Overview

This project analyzes an online retail dataset to understand sales trends, top products, and customer distribution.

---

 Dataset

* Source: Kaggle (Online Retail Dataset)
* Type: E-commerce transaction data
* Contains:

  * InvoiceNo
  * Description (Product)
  * Quantity
  * UnitPrice
  * InvoiceDate
  * Country

---

 Tools & Technologies

* Python
* Pandas
* Matplotlib

---

 Tasks Performed

* Data cleaning (handling missing values)
* Created new column: **TotalPrice**
* Extracted month from date
* Performed data analysis

---

Analysis & Results

### 1. Monthly Sales Trend

* Analyzed revenue month-wise
* Identified peak sales months

### 2. Top Selling Products

* Found top 5 most sold products

### 3. Country-wise Revenue

* Identified top countries generating revenue
